== Description ==

Ce plug-in a été fait pour se lier à la plateforme AuMenu.info.
C’est un service offert aux microbrasseries, bars et restaurants afin d’afficher ce qui se trouve présentement sur leurs lignes de bières en fût, leur menu de nourriture et plus.

Le plug-in affiche le contenu que vous créez sur AuMenu avec l’aide de notre SDK directement sur votre site Wordpress.
C’est simple, gérez vos lignes de bière en fût à un seul endroit et affichez-le partout où vous le voulez!

* créer une section avec la liste de vos bières que vous produisez à partir d’AuMenu
* créer une section avec la liste de vos bières présentement sur vos lignes à partir d’AuMenu
* créer une section pour votre menu de nourriture à partir d’AuMenu
* afficher un widget avec le contenu de chacune des sections du plug-in.

———————————————————————————————————————

This plugin is made to link with the platform named AuMenu.info It's a service offered to brew pubs to display their beers currently on taps, foods menu and more!

It display content you create into AuMenu with the use of our SDK directly on your WordPress site. It's simple, manage your taps list at only one place and display it everywhere you want it!

create a listing of your beers from AuMenu
create a listing of your beers on taps from AuMenu
create a food menu from AuMenu
create a widget for any AuMenu page



== Installation ==

Téléverser le plug-in AuMenu sur votre blogue, activé le, finalement entré votre clé d’API que vous trouverez dans votre panneau de contrôle AuMenu.info (https://www.aumenu.info/panel/).

———————————————————————————————————————

Upload the AuMenu plugin to your blog, Activate it, then enter your AuMenu.info API key (https://www.aumenu.info/panel/).
